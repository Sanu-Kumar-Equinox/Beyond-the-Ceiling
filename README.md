# Beyond the Ceiling

> A story-driven sci-fi dystopian adventure written in C++.

## Overview

**Beyond the Ceiling** is a narrative-focused text adventure set in a future where humanity lives beneath an artificial ceiling, governed by an all-powerful system that suppresses instability in the name of survival.

After a personal tragedy shatters your understanding of the world, you begin uncovering secrets hidden beneath decades of control, propaganda, and fear. What starts as a search for answers becomes a journey that forces you to question everything humanity believes about freedom, truth, and survival.

As the mystery unfolds, one question remains:

> **What lies beyond the ceiling?**

---

## Features

- Story-driven gameplay
- Interactive choices and branching dialogue
- Original sci-fi dystopian setting
- Atmospheric worldbuilding
- Mystery and exploration
- Character-focused narrative
- Terminal-style experience
- Built entirely in C++

---

## How to Play

Compile the project using a C++ compiler:

```bash
g++ main.cpp -o BeyondTheCeiling
```

Run the generated executable and follow the on-screen instructions.
